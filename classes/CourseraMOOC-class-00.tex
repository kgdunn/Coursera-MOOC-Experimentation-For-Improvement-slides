\begin{frame}\frametitle{How many experiments should I run?}
	For this course, a good start is to run $2^k$ experiments
	\begin{itemize}
		\item	Every factor ($k$) is at 2 levels
		\item	For example: $k$ = 3 factors, requires 8 experiments
	\end{itemize}
